# 租金查詢平台 — 產品規格

## 產品定位

基於內政部實價登錄公開資料的全台租金行情查詢工具，免費、免註冊。

## 目標用戶

- 租屋族：查詢目標區域的合理租金範圍
- 房東/包租代管業者：瞭解市場行情、坪效分析
- 研究者：租金趨勢、CPI 分析

## 核心功能

### 1. 多維度租金查詢
- **地理維度**: 縣市 → 區 → 路段（三層鑽取）
- **篩選維度**: 出租型態、坪數、樓層、租金範圍
- **路段搜尋**: 跨縣市全文搜尋路段名稱

### 2. 統計摘要
- 中位數/平均租金、資料筆數、平均坪數、每坪租金
- 年度同比變化（趨勢資料）
- 各房型分項比較

### 3. 視覺化
- 各區租金長條圖
- 歷年租金走勢折線圖
- Leaflet 互動地圖（熱力圖）
- CPI 住宅租金指數（指數/年增率/區域比較）

### 4. 進階分析
- 591 開價 vs 實價比較
- 出租型態分析（整戶/套房/雅房）
- 坪效分析（每坪租金比較）
- 區域生活機能評分（交通/宜居/人口/治安）
- 建物資訊（屋齡/電梯/管理員比例）

### 5. 社會住宅
- 社宅實價登錄（依區/依出租型態）
- 社宅 vs 市場租金比較
- 全國社宅現況統計
- 未來規劃（2032 政策目標）

## 資料來源

| 資料 | 來源 | 更新頻率 |
|------|------|----------|
| 租金統計 | 內政部實價登錄 2.0 | data-pipeline 產生 |
| CPI 指數 | 主計總處 | 年度 |
| 社宅資料 | 社會住宅推動聯盟 + 實價登錄 | data-pipeline 產生 |
| 591 開價 | 591 租屋網 | data-pipeline 爬取 |
| 區域評分 | 多來源綜合 | data-pipeline 產生 |

## 非功能需求

- **效能**: 首頁載入 < 3 秒（靜態 JSON）
- **響應式**: 手機/平板/桌面均可正常使用
- **無障礙**: 基本鍵盤導航支援
- **SEO**: SSR 預設啟用
